# AMDMServer
### IoT장비 데이터 전송
1. 데이터 교환 형태
>* type : 기기의 형태
>* id : 기기의 아이디(초기 설정때 서버에서 지급)
>* Lock : 현재 기기의 잠금 유무
>* PhoneLock : 기기안의 핸드폰의 잠금 상태

```
{
        "type": "IoT",
        "id": 1234567,
        "Lock": 0,
        "PhoneLock": {
                "19-760730001": 0,
                "19-760730004": 0,
                "19-760730003": 1,
                "19-760730002": 0
        }
}
```
